# Task Tracker
## How to use:

To use the application, first open the file `multipage_test.html` in a web browser, such as Chrome, Firefox, or Edge. In other words, copy-paste the `multipage_test.html` file's location into a new tab. Then, you may bookmark the site on your browser if you wish.

---

## Menu and Features

To begin using your site, start by clicking one of the numbers to open a page. The title (which displays "New File" by default) and the content ("New content") could be edited freely.

### Saving Content

There are two ways to save content written in either of the editable boxes. For example, the **Save Page** button at the bottom of the page does so.

Alternatively, the shortcut `Ctrl+S` (`Command+S` on Mac) does the same thing, and is recommended for PC users.

### Buttons

There are various buttons on the page in darker green boxes. Most of these buttons have reactions.

#### Arrow Buttons

The arrow buttons on the top of the page ("**<**" and "**>**") help navigate between pages. Going below Page 1 cycles you to the last page by default. Furthermore, going beyond the last page (by default 5) prompts you to create another page. 

#### Open Menu

Simply reopens the menu introduced from loading the page.

#### Add List

Adds a list into the content box. Lists do not support indentation as of now, but are still useful for organizing.

#### Change CSS

Changes the page's display into a different color between green, blue, and red.
